# Panchangam — Domain Knowledge

Panchangam (పంచాంగం) means "five limbs" in Sanskrit/Telugu.
The five elements calculated for each day:

| # | Name | Telugu | Description |
|---|------|--------|-------------|
| 1 | Tithi | తిథి | Lunar day (30 per lunar month) |
| 2 | Vara | వారం | Day of the week (with ruling deity) |
| 3 | Nakshatra | నక్షత్రం | Lunar mansion / star (27 total) |
| 4 | Yoga | యోగం | Auspicious/inauspicious combination (27 total) |
| 5 | Karana | కరణం | Half of a Tithi (11 types) |

## Additional Elements (varies by tradition)
- Rahu Kalam — inauspicious time period each day
- Gulika Kalam — another inauspicious period
- Yamaganda Kalam — third inauspicious period
- Abhijit Muhurtham — auspicious midday window
- Sunrise / Sunset times
- Masa (Telugu month)
- Samvatsara (Telugu year name, 60-year cycle)
- Ayanam (Uttarayana / Dakshinayana)
- Ruthuvu (Season)

## Calculation Basis
- Based on Hindu astronomical system (Surya Siddhanta)
- Location-dependent (sunrise/sunset varies by geography)
- Telugu calendar follows Amavasyant or Purnimant — need to confirm which tradition

## Notes
- All calculations must be accurate for the Telugu-speaking audience
- Primary audience: Telugu Hindus in India and diaspora
